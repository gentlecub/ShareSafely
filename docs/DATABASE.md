# Base de Datos

## Tablas sugeridas

### Archivos
- ID
- Nombre
- Fecha de subida
- Tamaño
- Tipo
- Estado

### Enlaces
- ID
- Archivo asociado
- Token SAS
- Fecha de expiración
- Estado (Activo / Expirado / Eliminado)

### Logs de acceso
- ID
- Archivo
- Usuario (si aplica)
- Acción (Descarga / Subida)
- Timestamp
