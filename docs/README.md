# Plataforma de Gestión de Archivos en Azure

## Objetivo
Crear una plataforma que permita:
- Subir archivos a Azure Blob Storage
- Generar un enlace seguro, único y con expiración
- Permitir descarga solo mientras el enlace esté activo
- Proteger todas las credenciales usando Azure Key Vault
- Limpiar archivos expirados automáticamente

## Tecnologías Obligatorias
- Backend: ASP.NET Core Web App o Web API
- Frontend: HTML/CSS/JavaScript o React
- Cloud Hosting: Azure Web Apps
- Almacenamiento: Azure Blob Storage
- Seguridad: Azure Key Vault
- Automatización: Azure Functions o Logic Apps
- Monitoreo: Azure Monitor / Application Insights
